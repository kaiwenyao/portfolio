# Kaiwen Yaos Portfolio

AstroとTailwind CSSで構築された个人ポートフォリオサイト。

[English](./README.md)
[中文](./README_zh.md)
[日本語](./README_ja.md)

## 技術スタック

- **フレームワーク:** Astro
- **スタイリング:** Tailwind CSS
- **言語:** TypeScript

## 始め方

```sh
pnpm install
pnpm dev
```

## コマンド

| コマンド        | 操作                               |
| :------------- | :-------------------------------- |
| pnpm dev       | :4321 ポートで開発サーバーを起動      |
| pnpm build     | ./dist/ に本番用サイトをビルド       |
| pnpm preview   | ローカルでビルドをプレビュー          |
